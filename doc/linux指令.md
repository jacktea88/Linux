
### 一、 指令下達基本語法
在 Linux 中，指令的基本結構為：**`command [options] [arguments]`**。
*   **Command：** 指令名稱。
*   **Options：** 選項（如 `-a` 或 `--help`），用來調整指令行為。
*   **Arguments：** 參數，指指令作用的對象（如檔案或目錄名稱）。

### 二、 系統入門與輔助按鍵
*   **常用按鍵：**
    *   **Tab：** 指令或檔案名稱補全。
    *   **Ctrl + C：** 中斷目前正在前景執行的程式。
    *   **Ctrl + D：** 中斷 Shell 互動模式或登出。
    *   **↑ ↓：** 切換已使用過的指令歷史。
*   **求助與結束：**
    *   **man：** 查看指令的詳細使用手冊（如 `man cat`）。
    *   **exit / logout：** 登出系統。
    *   **shutdown now / init 0：** 立即關機。
    *   **reboot / init 6：** 重新開機。

### 三、 目錄與檔案基本操作
*   **路徑切換與檢視：**
    *   **pwd：** 顯示目前所在的工作目錄。
    *   **cd：** 變換目錄（`~` 代表家目錄，`..` 代表上一層）。
    *   **ls：** 檢視檔案與目錄。常見選項有 `-a` (顯示隱藏檔)、`-l` (詳細資料)。
*   **目錄管理：**
    *   **mkdir：** 建立新目錄。`-p` 可遞迴建立多層目錄。
    *   **rmdir：** 刪除「空」的目錄。
*   **檔案管理：**
    *   **touch：** 建立空檔案或更新時間戳記。
    *   **cp：** 複製檔案或目錄（`-r` 用於目錄，`-a` 保留屬性）。
    *   **mv：** 移動或更名檔案與目錄。
    *   **rm：** 刪除檔案或目錄。`-i` 提示確認，`-rf` 強制刪除目錄。

### 四、 檔案內容查看與編輯
*   **內容瀏覽：**
    *   **cat：** 瀏覽並列印檔案內容（`-n` 顯示行號）。
    *   **tac：** 反向列印檔案內容。
    *   **nl：** 列印並標示行號。
*   **文字編輯：**
    *   **vi / vim：** 強大的傳統編輯器，包含標準、編輯與指令列三種模式。
    *   **nano：** 簡易的文字編輯器。

### 五、 資料流重導向與管線 (Pipeline)
*   **重導向符號：**
    *   **`>` / `>>`：** 將標準輸出 (stdout) 覆蓋或附加到檔案。
    *   **`2>`：** 將標準錯誤輸出 (stderr) 重導到檔案。
    *   **`<`：** 以檔案內容作為標準輸入 (stdin)。
*   **管線符號 `|`：**
    *   將前一個指令的輸出作為後一個指令的輸入（例如：`ls | grep 'test'`）。
*   **常見過濾器指令：**
    *   **grep：** 搜尋特定的字串或樣式。
    *   **tee：** 同時將結果輸出至螢幕與檔案。

### 六、 帳號與權限管理
*   **身分與權限變更：**
    *   **sudo：** 以管理員 (root) 身分執行指令。
    *   **su -：** 切換使用者身分並套用其環境變數。
    *   **chmod：** 改變存取權限（數字模式如 755，符號模式如 u+x）。
    *   **chown：** 改變檔案的擁有者及擁有群組。
*   **帳號管理：**
    *   **useradd / userdel：** 新增或刪除使用者。
    *   **passwd：** 修改使用者密碼。
    *   **id / groups：** 查詢使用者身分與群組資訊。

### 七、 系統監控與程序管理
*   **磁碟與系統資訊：**
    *   **df -Th：** 檢視檔案系統的磁碟空間使用現況。
    *   **uname -a：** 取得系統核心資訊。
    *   **uptime：** 顯示系統運行時間與登入人數。
*   **程序監控：**
    *   **ps -aux：** 觀察系統所有程序資料。
    *   **top：** 動態監視系統資源與程序。
    *   **kill：** 中止程序。`-9` 為立刻強制停止，`-2` 等同 Ctrl+C。

### 八、 檔案備份與壓縮
*   **tar：** 打包與壓縮指令。
    *   **`-zcvf`：** 使用 gzip 壓縮檔案。
    *   **`-jcvf`：** 使用 bzip2 壓縮檔案。
    *   **`-zxvf`：** 解壓縮檔案。
